For development on windows
--------------------------
Assuming, you unzipped the zip file to c:\workspace\. You will have the following folder
c:\workspace\techsolutions

1. Install 'cmake.exe' from
2. Create a directory techsolutionswin under c:\workspace and 'cd' to that directory.
3. Invoke cmake.exe as follows,
  c:\workspace\techsolutionswin> cmake.exe -G"Visual Studio 12" ..\techsolutions\
4. You should have a visual studio project now that you can open in Visual studio and start coding.
